# 项目上线准备清单

> 针对 Phase 1~4 全部功能，基于阿里云体系，按优先级排列。
> 标记说明：🔴 必须（阻塞开发）| 🟡 重要（阻塞上线）| 🟢 可后补（不阻塞 MVP）

## 一、AI 模型 API（🔴 最优先）

三个模型同时使用，均衡分配读者，开发第一天就需要。

### 1. DeepSeek（~34% 读者）

- [ ] 注册 DeepSeek 开放平台账号
- [ ] 获取 API Key
- [ ] 确认使用模型：`deepseek-chat`（即 DeepSeek V3）
- [ ] 确认价格：输入 ¥1/百万token，输出 ¥2/百万token
- [ ] 充值余额（建议先充 ¥50 用于开发测试）
- [ ] 确认并发限制（QPM/TPM），如不够需申请提额
- **控制台**：https://platform.deepseek.com
- **API 格式**：兼容 OpenAI Chat Completions
- **Base URL**：`https://api.deepseek.com`

### 2. 通义千问 Qwen（~33% 读者）

- [ ] 注册阿里云账号（你应该已有）
- [ ] 开通"模型服务灵积 DashScope"
- [ ] 获取 API Key
- [ ] 确认使用模型：`qwen-plus`（性价比最优）
- [ ] 确认价格：输入 ¥0.8/百万token，输出 ¥2/百万token
- [ ] 查看免费额度（新用户通常有赠送）
- [ ] 确认并发限制，如不够申请提额
- **控制台**：https://dashscope.console.aliyun.com
- **API 格式**：兼容 OpenAI Chat Completions
- **Base URL**：`https://dashscope.aliyuncs.com/compatible-mode/v1`

### 3. 智谱 GLM（~33% 读者）

- [ ] 注册智谱 AI 开放平台账号
- [ ] 获取 API Key
- [ ] 确认使用模型：`glm-4-flash`（免费）
- [ ] 确认免费额度和限制（QPM 限制可能较严）
- [ ] 如免费额度不够，考虑 `glm-4-air`（低价）
- [ ] 确认并发限制
- **控制台**：https://open.bigmodel.cn
- **API 格式**：兼容 OpenAI Chat Completions
- **Base URL**：`https://open.bigmodel.cn/api/paas/v4`

### 需要记录的环境变量

```env
DEEPSEEK_API_KEY=sk-xxx
DEEPSEEK_BASE_URL=https://api.deepseek.com

DASHSCOPE_API_KEY=sk-xxx
DASHSCOPE_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1

ZHIPU_API_KEY=xxx.xxx
ZHIPU_BASE_URL=https://open.bigmodel.cn/api/paas/v4
```

## 二、阿里云服务器 ECS（🔴 部署必须）

### 购买配置

- [ ] 购买 ECS 实例
- **推荐规格**：2 核 4G（ecs.c6.large 或 ecs.s6-c1m2.large）
- **系统**：Ubuntu 22.04 LTS 或 Alibaba Cloud Linux 3
- **磁盘**：40G SSD 云盘（系统盘）
- **带宽**：5Mbps 按固定带宽（或按量，初期流量不大）
- **地域**：华东（上海）或华北（北京）
- **购买时长**：包年更划算（~¥100/月，包年约 ¥800~1000/年）

### 服务器初始化

- [ ] 设置 root 密码或 SSH 密钥对
- [ ] 配置安全组规则：
  - 入方向开放：22（SSH）、80（HTTP）、443（HTTPS）、3000（Next.js 开发）
- [ ] 安装基础环境：
  - Docker + Docker Compose
  - Node.js 20 LTS（通过 nvm）
  - Nginx（反向代理）
  - Git
- [ ] 配置 swap 分区（2G，4G 内存有时不够）

### 月费预估：~¥100/月

---

## 三、阿里云 RDS MySQL（🔴 开发必须）

### 购买配置

- [ ] 购买 RDS MySQL 实例
- **版本**：MySQL 8.0
- **规格**：1 核 2G 基础版（`mysql.n1.micro.1`）— 初期够用
- **存储**：20G SSD（按量扩容）
- **地域**：与 ECS 同地域（内网互通，免流量费）

### 实例配置

- [ ] 创建数据库：`novel_eval`
- [ ] 创建数据库账号（非 root，用于应用连接）
- [ ] 设置白名单：添加 ECS 内网 IP
- [ ] 记录连接信息

### 需要记录的环境变量

```env
DATABASE_URL=mysql://用户名:密码@内网地址:3306/novel_eval
```

### 月费预估：~¥80/月

> **开发阶段替代方案**：本地用 Docker 跑 MySQL，不用先买 RDS。`docker-compose.yml` 里已包含。

## 五、阿里云 OSS 对象存储（🟡 报告存储）

### 开通配置

- [ ] 开通 OSS 服务（通常阿里云账号默认已开通）
- [ ] 创建 Bucket
  - 名称：`novel-eval-reports`（或你喜欢的名字）
  - 地域：与 ECS 同地域
  - 存储类型：标准存储
  - 读写权限：私有（通过签名 URL 访问）
- [ ] 创建 RAM 子账号，授予 OSS 读写权限
- [ ] 获取 AccessKey ID 和 AccessKey Secret

### 需要记录的环境变量

```env
ALIYUN_OSS_ACCESS_KEY_ID=xxx
ALIYUN_OSS_ACCESS_KEY_SECRET=xxx
ALIYUN_OSS_BUCKET=novel-eval-reports
ALIYUN_OSS_REGION=oss-cn-shanghai
```

### 月费预估：~¥10/月（按量计费，初期很便宜）

## 六、域名 + ICP 备案（🟡 上线必须）

### 域名

- [ ] 购买域名（阿里云万网）
  - 建议 `.com` 或 `.cn`
  - 示例：`aireader.cn`、`noveleval.com`、`ai-shidu.com` 等
- [ ] 完成域名实名认证

### ICP 备案

- [ ] 在阿里云提交 ICP 备案申请
  - **个人备案**：需身份证正反面、手持身份证照片
  - **企业备案**：需营业执照、法人身份证
- [ ] 备案审核周期：通常 **7~20 个工作日**
- [ ] 备案期间可以用 ECS 公网 IP 直接访问做开发测试

> ⚠️ **备案是最耗时的环节，建议第一天就提交！**

### HTTPS 证书

- [ ] 阿里云提供免费 DV SSL 证书（单域名，1年有效）
- [ ] 或使用 Let's Encrypt（免费，需定期续期）
- [ ] 在 Nginx 中配置 SSL

### DNS 解析

- [ ] 在阿里云 DNS 控制台添加 A 记录，指向 ECS 公网 IP

### 费用：域名 ~¥50/年，证书免费

## 九、本地开发环境（🔴 最优先）

### 基础工具

- [ ] Node.js 20 LTS（建议用 nvm 管理）
- [ ] pnpm（推荐）或 npm/yarn
- [ ] Git
- [ ] Docker Desktop（用于本地跑 MySQL + Redis）
- [ ] VS Code / Cursor（你已有）

### Docker 本地开发

- [ ] 准备 `docker-compose.yml`（项目中已规划）

```yaml
# docker-compose.yml（本地开发用）
version: '3.8'
services:
  mysql:
    image: mysql:8.0
    ports:
      - "3306:3306"
    environment:
      MYSQL_ROOT_PASSWORD: root123
      MYSQL_DATABASE: novel_eval
    volumes:
      - mysql_data:/var/lib/mysql

  redis:
    image: redis:7-alpine
    ports:
      - "6379:6379"

volumes:
  mysql_data:
```

### 本地环境变量文件

- [ ] 创建 `.env.local`

```env
# 数据库（本地 Docker）
DATABASE_URL=mysql://root:root123@localhost:3306/novel_eval

# Redis（本地 Docker）
REDIS_URL=redis://localhost:6379

# AI 模型 API
DEEPSEEK_API_KEY=sk-xxx
DEEPSEEK_BASE_URL=https://api.deepseek.com
DASHSCOPE_API_KEY=sk-xxx
DASHSCOPE_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
ZHIPU_API_KEY=xxx.xxx
ZHIPU_BASE_URL=https://open.bigmodel.cn/api/paas/v4

# 应用配置
NEXT_PUBLIC_APP_URL=http://localhost:3000
```
